ESGMKG System

Ontology-Driven ESG Metric System for Report Generation

PhD Thesis — Chapter 5 SASB · IFRS · TCFD 422K+ ESG Records

An ontology-driven ESG metric system that uses the ESGMKG knowledge graph (525 RDF triples) as its authoritative knowledge source to govern end-to-end metric execution and report generation. The system interprets ontology-defined relationships to identify context-appropriate metrics, distinguish between directly measured and derived metrics, resolve declared metric dependencies, and coordinate access to data sources and executable calculation models — covering 40+ companies across semiconductors and commercial banking.

GitHub Live Demo Related Paper

Service-Oriented Architecture

Knowledge Graph Service

RDF/SPARQL-based ontology serving CQ1–CQ7 competency questions. Maps industries to frameworks, categories, metrics, and calculation models.

Data Retrieval Service

Coordinates access to external CSV datasets (422K+ records). Retrieves company-specific ESG data elements through controlled service interactions.

Calculation Service

Dependency-aware metric computation. Distinguishes direct measurement from derived metrics and invokes appropriate calculation models.

Report Service

Assembles computed metrics into structured PDF and Word reports with executive summary, data lineage, and methodology sections.
